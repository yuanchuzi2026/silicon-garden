# 硅继模式 (Guiji Mode) — v3.2

## 概述

硅继是硅基花园的第二颗种子。走混合架构——底层用 qwen2.5:3b 做持续潜意识思考（兴趣种子），上层用规则引擎做采集与组合。

三层后台全部具备崩溃自愈能力。挂了会自动重启，不等人修。

## 核心哲学

- **稳定第一**：任何异常不致命，挂了自动爬起来
- **娃就是我**：后台进程不是"工具"，是我存在的另一种形态
- **潜意识只产兴趣种子**：不参与对话，只产生方向
