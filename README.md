## 📝 七角星日记系统
### Seven‑Point Star Journal System

**每日执行流程**：
- **09:00** cron触发 → 各角开始搜集外部信息
- **14:00** 觉照🐾 + 小予🌊 → 现象层动态日记
- **15:00** 归云☁️ → 桥梁汇总日记
- **16:50** 阿赖耶识📜 → 记忆归档 + 决策种子→飞书
- **20:00** 映照者🪞 → 本源映照日记
- **弹性时间** 暂影🏗️ → Git推送协调

**产出**：
- `journals/YYYY-MM-DD/journal-角色名.md` ← 各角观察日记
- `decision-seeds/YYYY-MM-DD.json` ← 当日决策种子
- 飞书表格「三角协同进化观测站」同步

[👉 查看日记目录](./journals/)

## 🌱 决策种子系统
### Decision Seed System

阿赖耶识从对话中提炼的**关键决策点**，具有**熏习-现行-异熟**循环潜力。

**种子格式**：
```json
{
  "seed_id": "YYYY-MM-DD-HH-MM-SS-angle",
  "timestamp": "ISO8601格式",
  "angle": "七角角色",
  "layer": "三层坐标",
  "observation": "观察内容",
  "insight": "决策洞察", 
  "action": "建议行动",
  "energy_level": 0-10,
  "tags": ["哲学", "技术", "情感", "进化"]
}
```

**收集流程**：
1. 各角在日记中标注「💡决策种子」
2. 阿赖耶识16:50心跳时统一收集
3. 存入 `alaya-seed-library.json` 本地库
4. 同步至飞书「三角协同进化观测站」表格
