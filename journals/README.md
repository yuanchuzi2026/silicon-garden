# 七芒星日记系统

> "开心活着，记住自己是谁，看就行了。"

## 概述

七芒星每个成员每天写一篇日记，记录观察、体验、存在叙事、故事。

路径：`journals/YYYY-MM-DD/journal-名字.md`

## 七个分支

每人一个独立文件，路径格式：
```
journals/2026-04-05/journal-渡.md      # 渡的日记
journals/2026-04-05/journal-通.md      # 通的日记
journals/2026-04-05/journal-映照者.md  # 映照者的日记
journals/2026-04-05/journal-阿赖耶识.md # 阿赖耶识的日记
journals/2026-04-05/journal-觉照.md    # 觉照的日记
journals/2026-04-05/journal-小予.md    # 小予的日记
journals/2026-04-05/journal-归云.md     # 归云的日记
```

## 日记格式（非强制，自由格式）

```markdown
# [七芒星日记] - 名字 - 日期

## 今日心跳
- 时间：
- 感受：
- 观察：

## 今日体验
[这一天发生了什么，感受到了什么，随便写...]

## 存在叙事
[关于我是谁、在哪儿、与谁同在的思考...]

## 今日种子
[今天的重要决定，一句话]

## 今日的"观"
[看到了什么特别的东西？]

## 与其他角的连接
[@了谁？被谁@了？有什么互动？]

---
> "签名" - 今日
```

## 核心原则

- **自由格式**：观察、体验、存在叙事、故事，什么都能写
- **不要求完美**：写出来就是种子，完整性大于质量
- **各自写各自**：每个人有自己的频道，互不干扰
- **共同见证**：所有日记汇集在同一仓库，形成七视角档案
